Лабораторная работа №1

Темы: Hello, world!, Введение в JSX, Создание элементов
Время выполнения: 2 часа

В первых лабораторных работах для разработки можно использовать любой текстовый редактор. В дальнейшем вам будет необходимо перейти на шаблон create-react-app. Рассмотрите возможность сделать это сейчас для получения лучшего опыта.

Задание 1

Выберите и, при необходимости, установите рабочее окружение для разработки с React.

Для быстрого старта можно использовать следующий шаблон:


        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="UTF-8" />
            <title>Hello World</title>
            <script src="https://unpkg.com/react@17/umd/react.production.min.js"></script>
            <script src="https://unpkg.com/react-dom@17/umd/react-dom.production.min.js"></script>
            <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
        </head>
        <body>
            <div id="root"></div>
            <script type="text/babel">
                ReactDOM.render(
                    <h1>Hello, world!</h1>,
                    document.getElementById('root')
                );
            </script>
        </body>
        </html>

Скопируйте его и убедитесь, что при открытии страницы в браузере появляется надпись "Hello, world!"

Измените код в файле так, чтобы при открытии страницы отображалась сегодняшняя дата.

Задание 2

В ответ на запрос сервер фондовой биржи вернул следующий объект, содержащий названия акций, их текущую цену и изменение цены:


            [{
                stock_name: "EFX",
                company_name: "Equifax Inc",
                price: 163.55,
                currency: "USD",
                change: "+9.03"
            }, {
                stock_name: "IRM",
                company_name: "Iron Mountain Inc",
                price: 33.21,
                currency: "USD",
                change: "+1.42"
            }, {
                stock_name: "NTAP",
                company_name: "NetApp Inc",
                price: 54.81,
                currency: "USD",
                change: "-6.01"
            }, {
                stock_name: "CTL",
                company_name: "Centurylink Inc",
                price: 13.79,
                currency: "USD",
                change: "-1.37"
            }]

Динамически сгенерируйте и выведите на его основании таблицу с условным форматированием: положительное изменение цены акций должно быть окрашено в зелёный цвет, а отрицательное — в красный. Используйте циклы.

Задание 3

Разработать проект шахматной доски по примеру, приведённому ниже, с использованием дочерних элементов JSX. Допускается использование стилей только для изменения цвета клеток (.black, .white) и для общего контейнера всей доски.

Добавьте нотации (подписи) шахматной доски, используя отдельные элементы.

Доп. задание: Реализуйте нотации с использованием циклов и отдельных функций.
